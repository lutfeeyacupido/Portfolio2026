# Lutfeeya Cupido | Portfolio Website
"Designing the Frontend, Securing the Backend". A modern, responsive portfolio showcasing my journey as an entry-level developer with a passion for design, photography, and web development.

# Features
Feature	Description
- Dual Theme	Toggle between Purple Mode and Green Mode with smooth transitions
- Fully Responsive	Optimized for desktop, tablet, and mobile devices
- Project Showcase	Interactive cards displaying key projects with GitHub links
- Certifications Grid	Clean display of technical certifications
- Contact Section	Quick access to email, phone, and LinkedIn
- Smooth Animations	Hover effects, floating icons, and scroll behaviors
- Theme Persistence	User's theme preference saved in localStorage

# User Guide
- Theme Toggle:
  - Location: Fixed button in the top-right corner
  - Click to switch between 💚 Green Mode and 🟣 Purple Mode
  - Your preference is automatically saved
- Navigation Buttons
  - Contact Me	Scrolls to Contact section with highlight
  - View Projects	Scrolls to Projects section with highlight
- Project Cards
  - Hover → Card lifts up with colored border
  - Click "🐙 View Source Code" → Opens GitHub repo in new tab
  - Feature tags → Hover for subtle animation
- Certifications
  - Hover over any certification → Sliding animation effect
- Contact Section
  Click	What Happens:
  - Email	Opens email client
  - Phone	Shows/prompts to call
  - LinkedIn	Opens LinkedIn profile

# Pro Tips
- Try both themes! Purple and Green modes create completely different moods.
- Hover over everything — cards, buttons, skills, and certifications all have responsive animations.
- Check out the project icons — they float gently for a polished effect.
- Bookmark the site — your theme preference will be remembered!
